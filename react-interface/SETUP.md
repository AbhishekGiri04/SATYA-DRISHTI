# Frontend Setup

## Environment Configuration

1. Copy `.env.example` to `.env`:
```bash
cp .env.example .env
```

2. Update `VITE_API_URL` if backend is running on different port:
```
VITE_API_URL=http://localhost:8001
```

## Installation

```bash
npm install
npm run dev
```

Frontend will run on: http://localhost:5173
